6 for (int right=0; right<s.length(); right++) {

📖 Algorithm — How it works
1
Use two pointers — left & right — to define a window over the string.
2
Move right forward each step. Check if s[right] is already in the HashSet.
3
No duplicate → Add s[right] to set. Update maxLength = max(maxLength, right−left+1).
4
Duplicate found! → Shrink window: remove chars from left until the duplicate is gone.
5
Remove the duplicate itself from left, advance left++, then add new s[right] to set.
6
Repeat until right reaches end. Return maxLength — the answer! 🎉
